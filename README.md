# What To Wear (WTW) 👗📱

A mobile wardrobe management app built with .NET MAUI, 
allowing users to digitise and organise their wardrobe.

## Features
- Capture or upload photos of clothing items
- Categorise clothes (Top, Bottom, Outerwear, Footwear, Accessories, Activewear)
- Men's and Women's wear preferences
- User profile with profile picture
- Outfit scheduling tab
- Tabbed navigation interface

## How to Run
1. Install Visual Studio 2022 with .NET MAUI workload
2. Clone the repo
3. Open `WTW.csproj` in Visual Studio
4. Select Android emulator or device
5. Click Run

## Project Structure
- `Pages/` — All app pages (Home, Wardrobe, Profile, Schedule)
- `Models/` — Data models (Product, User, ClothingPreference)
- `App.xaml` — App entry point
- `MauiProgram.cs` — App configuration
